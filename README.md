# 🛡️ SALVA Ecosystem 
> **Sovereign. Multichain. Vertically Integrated.**

Salva is a unified payments and identity infrastructure designed to bridge the gap between fragmented EVM networks. By vertically integrating Naming, Wallets, and Stablecoins, we provide a "Closed-Loop" financial experience that abstracts the underlying chain complexity.

---

## 💎 The Trinity
Salva is built on three core pillars that operate as a single, cohesive unit:

### 1. 🏷️ SNS (Salva Naming Service)
A high-efficiency, multichain registry using a **Factory-Clone (EIP-1167)** architecture.
* **Low Cost:** Deploy sub-registries cheaply on L2.
* **Optimized:** Custom `NameLib` with bit-packed storage and gas-efficient resolution.

### 2. 💳 Salva Smart Wallet
A next-generation Account Abstraction wallet that serves as the gateway to the ecosystem.
* **EIP-7702 Ready:** Native support for the latest Ethereum execution upgrades.
* **Gas-Free UX:** Built-in paymasters for seamless NGN transactions.

### 3. 🪙 NGN Stablecoin
The settlement layer for Nigerians in the Salva ecosystem.
* **Native Utility:** Designed for direct integration with SNS handles.
* **Multichain Liquidity:** Consistent value across L1 and all supported L2s.
